# 🎓 Student Score Prediction

A Machine Learning project that predicts student exam scores based on hours studied using Linear Regression.

## 🚀 Technologies

- Python
- Pandas
- Scikit-Learn

## 📊 Dataset

Columns:

- hours_studied
- score

## 📚 Concepts Used

- Features & Target
- Train/Test Split
- Linear Regression
- Predictions
- MAE
- MSE
- R² Score

## ▶️ Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python main.py
```

## 🎯 Features

- Predict student scores
- User input predictions
- Model evaluation
- Slope & Intercept analysis
