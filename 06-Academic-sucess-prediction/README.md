# 🌲 Academic Success Predictor

A Machine Learning classification project that predicts whether a student will pass or fail based on academic performance metrics.

---

## 📌 Project Overview

This project uses the Random Forest Classifier algorithm to analyze student data and predict academic success.

The model learns patterns from historical student records and predicts whether a new student is likely to pass or fail.

---

## 🎯 Features

- Load student dataset
- Train/Test Split
- Random Forest Classification
- Accuracy Evaluation
- Student Pass/Fail Prediction
- Data Visualization using Matplotlib

---

## 📊 Dataset Columns

| Column | Description |
|----------|------------|
| hours_studied | Number of hours studied |
| attendance | Attendance percentage |
| assignments_completed | Number of assignments completed |
| pass | Target variable (0 = Fail, 1 = Pass) |
