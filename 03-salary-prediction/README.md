# Salary Prediction System 💰

A beginner Machine Learning project that predicts employee salaries based on years of experience using Linear Regression.

## Features

* Load salary dataset from CSV
* Split data into training and testing sets
* Train a Linear Regression model
* Predict salaries based on years of experience
* Evaluate model performance using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * R² Score
* Accept user input for salary prediction
* Visualize data using:

  * Scatter Plot
  * Regression Line

---

## Dataset

The dataset contains:

| Column     | Description         |
| ---------- | ------------------- |
| experience | Years of experience |
| salary     | Employee salary     |

Example:

| experience | salary |
| ---------- | ------ |
| 1          | 25000  |
| 2          | 30000  |
| 3          | 35000  |

## Evaluation Metrics

### Mean Absolute Error (MAE)

Measures the average prediction error.

### Mean Squared Error (MSE)

Measures error while giving more weight to large mistakes.

### R² Score

Measures how well the model learned the relationship between experience and salary.

---

## Visualization

### Scatter Plot

Shows actual employee salary data.

### Regression Line

Shows the pattern learned by the Linear Regression model.
